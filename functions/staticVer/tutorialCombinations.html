GSAP
This code animates an element with the class ".box" over 2 seconds. It moves the element 200 pixels to the right and 100 pixels down. The "ease" property applies a bouncing easing effect to the animation.
A.K.AGreenSock Animation Platform
gsap.to(".box", { duration: 2, x: 200, y: 100, ease: "bounce" });
KEY FEATURES
  1. A plugin for GSAP. Assigns elements of the HTML to trigger the animation.
  2. A property of the Scroll Trigger plugin. Synchronizes all of the children in the container to move with the parent container.
  3. Another property of the Scroll Trigger plugin. Correlates animation speed with the scroll speed.
